# Trip Planner

Static calendar planner for trips, appointments, accommodation, family visits, and travel milestones.

## Publish on GitHub Pages

1. Create a GitHub repo.
2. Upload `index.html`, `styles.css`, and `app.js` to the repo root.
3. In GitHub, open **Settings > Pages**.
4. Set the source to the main branch and root folder.

The planner saves shared trip data in Supabase. Run `supabase-setup.sql` in the Supabase SQL Editor. The page uses a lightweight local unlock only: username `fulton`, password `visa2026`. This is only a casual gate, not real security.

The starter trip is set up for the June 14 to July 9 Sydney visa trip window.
